## Linked List vs. Doubly Linked List

| Method               | Linked List | Doubly Linked List |
| -------------------- | ----------- | ------------------ |
| push(value)          | O(1)        | O(1)               |
| `pop() `             | `O(n)`      | `O(1) `            |
| unshift(value)       | O(1)        | O(1)               |
| shift()              | O(1)        | O(1)               |
| get(index)           | O(n)        | O(n)               |
| set(index, value)    | O(n)        | O(n)               |
| insert(index, value) | O(n)        | O(n)               |
| remove(index)        | O(n)        | O(n)               |
| reverse()            | O(n)        | O(n)               |
